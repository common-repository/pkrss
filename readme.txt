== Description ==

Support all over the world Popular RSS news disaggregated data source.
Supports various regions around the world voice reading.

== Installation ==

1. Upload the `src`(pkrss) folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the Plugins menu in WordPress
1. Visit the plugin configuration screen under **WordPress Admin > Settings > PKRSS** and set your preferences
1. Create posts using the easy to use shortcode and watch the magic happen
